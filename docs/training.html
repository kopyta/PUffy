training#

This is a documentation of module training of PUffy package. This module collects tools to train the model by performing iterative classification based on logistic regression - the second step in two-step classification.

training.Train#

class puffy.training.Train[source]#

Class for training logistic regression models.

iterative_LR#

Train.iterative_LR(X_df, rn, p, u, s_label, show_results_flag=True, epoch=12)[source]#

Trains a logistic regression model iteratively with a specified number of iterations.

Parameters:

  • X_df (pd.DataFrame) – Features.

  • rn (pd.DataFrame) – Reliable negatives.

  • p (pd.DataFrame) – Labeled positives.

  • u (pd.DataFrame) – Unlabeled instances.

  • s_label (np.array) – Indicator variable of labeled positives.

  • epoch (int, optional, default=12) – Number of iterations.

Returns:

Trained logistic regression model.

Return type:

LogisticRegression

Example

>>> from puffy.labelling_methods import LabellingMethods
>>> from puffy.rn_methods import RnMethods
>>> from puffy.training import Train
>>> from sklearn.datasets import load_breast_cancer
>>>
>>> X, y = load_breast_cancer(as_frame=True, return_X_y=True)
>>>
>>> Labeler = LabellingMethods(probability=0.9)
>>> labels = Labeler.sar_lr_sigmoid(X,y)
>>>
>>> RnIdentifier = RnMethods()
>>> rn,l,u = RnIdentifier.kmeans(X,labels)
>>>
>>> coach = Train()
>>> model = coach.iterative_LR(X,rn,l,u,labels)
